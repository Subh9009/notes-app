# Notes App

A simple Notes App built using MERN Stack (MongoDB, Express, Node.js, HTML, CSS, JS)

## Features
- Add Notes
- Delete Notes
- Pin Notes
- Search Notes

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB Atlas

## Run Locally
- cd backend
- npm install
- node server.js
